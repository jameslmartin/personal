Hack Night Demo

29 Jan. 2015

On Thursday, January 29th, I gave a short demo at UNC Chapel Hill at the CS club's Hack Night about using IPython Notebook and Pandas to do quick data analysis. I looked at some sample data from Ookla's Netmetrics product. Afterwards, I answered questions and chatted with interested undergraduates. This post was entirely made with IPython Notebook.

Installation

First, we're going to download Anaconda, which is a nifty distribution of Python that contains everything we need. Be careful Googling for "anaconda," Nicki Minaj has a monopoly on the search results. ;)

  • After running the bash script or opening the .exe file, pay close attention to the license agreement - it's easy to say "no" by accident.
  • Choose to prepend your PATH variable with the anaconda/bin folder. This makes it much easier to launch IPython Notebook right away. If you have problems, stop me or look on with someone else.
  • Also, run conda update conda and download any available updates.

When finished, create a new folder under your Documents or where you keep code to hold our notebooks; name it what you please. I call mine "notebooks."

  • Open your flavor of terminal and navigate to your new notebooks folder.
  • Execute the ipython notebook command

If you installed everything correctly, your default browser should pop up with a dashboard!

Now, we're going to download some sample data to have something to play with.

  • Navigate to https://www.ookla.com/netmetrics and click "Download Sample Data." Remember where you save the zip folder.

I'm going to save it to a folder one directory above where my notebooks reside. Let's extract the folder and see what we have. You can run shell commands within cells using a "bang" (!).

Telling stories

So now we know what the data contains, let's think about what we want to find out... Say my carrier has been really slow on my phone and I don't think I'm getting very good network speeds when using cellular data. I want to shop for a new phone and can change carriers. Let's find out which carrier has the fastest average (mean) network speed!

In [155]: 
ofinterest = carrs.keys()

for carrier in ofinterest:
    # Slice out carrier data
    carrier_data = us_tests[us_tests['carrier_name'] == carrier]
    latency = np.asarray(carrier_data['latency'])
    download_speed_mbs = np.asarray(carrier_data['download_kbps'].apply(lambda x: np.float64(x)*0.008))
    # Let's repurpose our old dictionary to contain the information we want.
    count = carrs[carrier]
    carrs[carrier] = [np.mean(latency), np.mean(download_speed_mbs), count]
